ประเภทของกล้องฟิล์ม 

2 min read

       การแบ่งประเภทของกล้องฟิล์ม สามารถแบ่งโดยใช้เกณฑ์ในการแบ่งหลักๆ ได้ 3 ประเภทได้แก่

2.1) Format ของฟิล์มที่ใช้ หรือขนาดของฟิล์ม

    2.1.1) Small Format : หมายถึงกล้องที่ใช้ฟิล์มขนาดเล็ก เช่น 35mm หรือฟิล์มปกติทั่วไป
    2.1.2) Medium Format : เป็นกล้องที่ใช้ฟิล์ม Format 120 (ขนาด 6x6" หรือ 6x9" )
    2.1.3) Large Format : เป็นกล้องที่ใช้ฟิล์มขนาดใหญ่ อย่างเช่น 4x5 นิ้ว หรือ ใหญ่กว่านั้นซึ่งเราไม่ค่อยได้เห็นทั่วไป ใช้งานค่อนข้างลำบาก


ฟิล์มประเภท Small Format (35mm) ทางด้านซ้าย และ ฟิล์มประเภท Medium Format (120) ทางด้านขวา


2.2) ลักษณะการโฟกัสภาพ

    2.2.1) กล้องฟิล์ม RF ( Rangefinder ) : เป็นกล้องฟิล์มที่มีลักษณะการโฟกัส โดยมองผ่านช่องมอง ภาพ ซึ่งเป็นคนละช่องกับเลนส์รับภาพ
    การโฟกัส ทำใด้โดย ตัววัดระยะของกล้องควบคู่กับการ ปรับวงแหวนโฟกัสของเลนส์ โดยการโฟกัสนั้นจะมีลักษณะเฉพาะ ที่เรียกว่า ภาพซ้อน       
    2.2.2) กล้องฟิล์ม SLR ( Single Lens Reflex ) : เป็นกล้องฟิล์มที่จะมีลักษณะการโฟกัสผ่านเลนส์ กล่าวคือ ในช่องมองภาพ (Viewfinder)
    เราเห็นอย่างไร ภาพที่เราได้ก็จะเป็นอย่างนั้น




กล้องฟิล์ม RF ( Rangefinder ) ทางด้านซ้าย และ กล้องฟิล์ม SLR ( Single Lens Reflex ) ทางด้านขวา
